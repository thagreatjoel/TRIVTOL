<h1 align="center">TRIVTOL</h1>
<p align="center" > It is an open source 3D printed three motor vertical take off and landing UAV frame designed to combine the capabilities of the tricoupter with speed and effiency,</p>


<p align="center">
  <img src="https://github.com/user-attachments/assets/f3fb3c38-2c5f-4171-8882-f502068dd5a3" />
</p>


> [!NOTE]
> Unlike normal quadcopter, It uses three brushless motors and 2 servo to get vertical flight


# GOALS
- Desgin a lightweight frame 
- Reduce manufacturing cost
- Achieve effiecent transition between over and forward flight

> [!TIP]
> Using metal geared servos is more relablble for titlting
