# GOALS
- Desgin a lightweight frame 
- Reduce manufacturing cost
- Achieve effiecent transition between over and forward flight

> [!TIP]
> Using metal geared servos is more relablble for titlting


<h2>TILT MECHANISM</h2>

<img align="left" height="200" alt="Tilt Mechanism" src="https://github.com/user-attachments/assets/77ca9f26-20d4-434b-b710-0ca922f57068"/>

The tilt machanism enables the front tip of wing rotates between and vertical and horisontal positions for the VTOL machanism and forward flight. <br>
Each front motor is mounted on a rotating shaft. it rotates using high torque gear servo is connect by the flight controller
